# workflow-glb

Muc dich: hien thi model GLB truc tiep tren `index.html`.

## Thu muc
- `input-models/`: dat file `.glb` (co the luu them `.fbx` de tham khao).

## Chay nhanh
1. Copy file GLB vao `input-models/`.
2. Mo `../index.html` bang Live Server hoac static server.
3. Tren giao dien, chon nguon `GLB`.

## Luu y
- Nen giu 1 file GLB chinh de team de quan ly.
- Neu model lech vi tri, chinh `modelConfig` trong `../index.html`.
